# Whispering Pages – Software Quality Assurance Testing Project

## Project Overview

This repository contains the **manual and automated testing documentation** for **Whispering Pages**, a web-based book reading tracker application developed using ReactJS.

The purpose of this project was to validate the application's functionality, usability, and reliability through structured **Software Quality Assurance (SQA)** practices. The testing process covered core features such as book management, reading progress tracking, quote collection, and review management.

This project demonstrates practical experience in:

* Manual testing
* Test case design
* Defect identification
* Bug severity & priority classification
* Automation testing
* End-to-End testing
* Regression testing

---

## Application Under Test (AUT)

Whispering Pages is a book tracking application that allows users to:

* Add and manage books
* Track reading progress
* Save memorable quotes
* Write and manage book reviews

---

### 1. Add Book Module

### 4. Book Reviews Module

## Defects Identified

### Add Book

* UI alignment issues
* Invalid title handling
* Inconsistent font styling

### Reading Progress

* Invalid page count handling
* Missing confirmation prompts
* Late validation triggering

### Quote Collection

* Character limit feedback missing
* Date highlighting issues

### Book Reviews

* Missing error messages for long comments
* Missing deletion confirmation
* UI alignment issues
